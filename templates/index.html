Network Metrics Explained
RTT (Round Trip Time)

Time for a packet to travel from source to destination and back. Lower is better.

Jitter

Variation in latency. High jitter causes unstable connections. Should be <10ms for VoIP.

Packet Loss

Percentage of packets that fail to reach destination. Should be <1% for good quality.

Bandwidth Estimate

Estimated throughput based on RTT and TCP window size (BDP calculation).

Protocol

ICMP = True ping (requires admin). TCP = Connection-based test (ports 80/443).

Network Quality

Overall status based on latency, jitter, and packet loss combined.
